Device-Tree bindings for hdmiphy driver

Required properties:
- compatible: value should be one of the following:
	1) "samsung,exyyess5-hdmiphy" <DEPRECATED>
	2) "samsung,exyyess4210-hdmiphy".
	3) "samsung,exyyess4212-hdmiphy".
- reg: I2C address of the hdmiphy device.

Example:

	hdmiphy {
		compatible = "samsung,exyyess4210-hdmiphy";
		reg = <0x38>;
	};
