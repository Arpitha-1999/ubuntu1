Samsung S6D16D0 4" 864x480 AMOLED panel

Required properties:
  - compatible: should be:
    "samsung,s6d16d0",
  - reg: the virtual channel number of a DSI peripheral
  - vdd1-supply: I/O voltage supply
  - reset-gpios: a GPIO spec for the reset pin (active low)

The device yesde can contain one 'port' child yesde with one child
'endpoint' yesde, according to the bindings defined in
media/video-interfaces.txt. This yesde should describe panel's video bus.

Example:
&dsi {
	...

	panel@0 {
		compatible = "samsung,s6d16d0";
		reg = <0>;
		vdd1-supply = <&foo>;
		reset-gpios = <&foo_gpio 0 GPIO_ACTIVE_LOW>;

		port {
			panel_in: endpoint {
				remote-endpoint = <&dsi_out>;
			};
		};
	};
};
