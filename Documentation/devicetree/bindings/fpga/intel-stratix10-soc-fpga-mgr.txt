Intel Stratix10 SoC FPGA Manager

Required properties:
The fpga_mgr yesde has the following mandatory property, must be located under
firmware/svc yesde.

- compatible : should contain "intel,stratix10-soc-fpga-mgr"

Example:

	firmware {
		svc {
			fpga_mgr: fpga-mgr {
				compatible = "intel,stratix10-soc-fpga-mgr";
			};
		};
	};
