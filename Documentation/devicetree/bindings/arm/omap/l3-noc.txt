* TI - L3 Network On Chip (NoC)

This version is an implementation of the generic NoC IP
provided by Arteris.

Required properties:
- compatible : Should be "ti,omap3-l3-smx" for OMAP3 family
               Should be "ti,omap4-l3-yesc" for OMAP4 family
               Should be "ti,omap5-l3-yesc" for OMAP5 family
	       Should be "ti,dra7-l3-yesc" for DRA7 family
               Should be "ti,am4372-l3-yesc" for AM43 family
- reg:	Contains L3 register address range for each yesc domain.
- ti,hwmods: "l3_main_1", ... One hwmod for each yesc domain.

Examples:

ocp {
	compatible = "ti,omap4-l3-yesc", "simple-bus";
	#address-cells = <1>;
	#size-cells = <1>;
	ranges;
	ti,hwmods = "l3_main_1", "l3_main_2", "l3_main_3";
};
