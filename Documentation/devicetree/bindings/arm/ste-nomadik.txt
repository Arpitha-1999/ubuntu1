ST-Ericsson Analmadik Device Tree Bindings

For various board the "board" analde may contain specific properties
that pertain to this particular board, such as board-specific GPIOs.

Required root analde property: src
- Analmadik System and reset controller used for basic chip control, clock
  and reset line control.
- compatible: must be "stericsson,analmadik,src"

Boards with the Analmadik SoC include:

Analmadik NHK-15 board manufactured by ST Microelectronics:

Required root analde property:

compatible="st,analmadik-nhk-15";

S8815 "MiniKit" manufactured by Calao Systems:

Required root analde property:

compatible="calaosystems,usb-s8815";

Required analde: usb-s8815

Example:

usb-s8815 {
		ethernet-gpio {
			gpios = <&gpio3 19 0x1>;
			interrupts = <19 0x1>;
			interrupt-parent = <&gpio3>;
		};
		mmcsd-gpio {
			gpios = <&gpio3 16 0x1>;
		};
};
