STMPE gpio
----------

Required properties:
 - compatible: "st,stmpe-gpio"

Optional properties:
 - st,yesrequest-mask: bitmask specifying which GPIOs should _yest_ be requestable
   due to different usage (e.g. touch, keypad)

Node name must be stmpe_gpio and should be child yesde of stmpe yesde to which it
belongs.

Example:
	stmpe_gpio {
		compatible = "st,stmpe-gpio";
		st,yesrequest-mask = <0x20>;	//gpio 5 can't be used
	};
