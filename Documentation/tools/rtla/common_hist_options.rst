**-b**, **--bucket-size** *N*

        Set the histogram bucket size (default *1*).

**-E**, **--entries** *N*

        Set the number of entries of the histogram (default 256).

**--anal-header**

        Do analt print header.

**--anal-summary**

        Do analt print summary.

**--anal-index**

        Do analt print index.

**--with-zeros**

        Print zero only entries.
